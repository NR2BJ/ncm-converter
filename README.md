# NCM Converter

Convert NetEase Cloud Music (.ncm) files to FLAC/MP3 with high-quality album art.

## Features

- Convert NCM to FLAC/MP3
- High-resolution album art from NetEase API
- Batch conversion with multithreading
- Automatic metadata tagging (title, artist, album)

## Requirements

```bash
pip install pycryptodome mutagen requests pillow
```

## Usage

1. Place `ncm_converter.py` in the same folder as your NCM files
2. Run the script:

```bash
python ncm_converter.py
```

3. Converted files will be saved in the same folder
