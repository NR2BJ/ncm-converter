# NCM Converter

Convert NetEase Cloud Music (`.ncm`) files to FLAC / MP3 with album cover.

**Web:** [nr2bj.github.io/ncm-converter](https://nr2bj.github.io/ncm-converter/) — browser-based, local, no upload.

**CLI:** `python ncm_converter.py` in a folder with `.ncm` files.

```
pip install pycryptodome mutagen requests pillow
```
