What is an NCM file?

NCM is a proprietary encrypted audio format used by NetEase Cloud Music (网易云音乐 / 넷이즈 클라우드 뮤직), one of the largest music streaming platforms in China. Files downloaded for offline listening are saved as .ncm and cannot be played by standard audio players.

NCM Converter decrypts the container, extracts the original audio stream (FLAC or MP3), fetches album artwork, and outputs a standard audio file — all within your browser. No server upload, no installation required.

How to convert NCM files

1. Open ncm.nr2bj.com
2. Drag and drop your .ncm files (or click to select)
3. Wait for conversion to complete
4. Download individual files or all at once as ZIP

Python CLI version

Prefer command-line? The Python script supports batch conversion with multithreading. Install dependencies with pip install pycryptodome mutagen requests pillow and run python ncm_converter.py in the folder with your NCM files.
